R2 home:


En la página principal de bienvenida, titulada como página "home", se muestra una métrica de R2 del modelo.
Esta métrica es calculada comparando la predicción que realiza el modelo del dataset correspondiente
año anterior al corriente con los datos reales de ese mismo dataframe.



Características de predicción individual:


Las características a rellenar en la predicción individual son las Features
que usa el modelo para predecir el precio de la vivienda unifamiliar. Estas features son las más representativas
del dataset y las que más peso tienen en el modelo.



Formato dataset de predicción csv:


Para poder predecir utilizando csv mediante la predicción por csv,
el dataset debe tener las siguientes columnas en el siguiente orden presentado:


{% for col in columns.columns %} {% endfor %} {% for row in columns.iterrows() %} {% for value in row[1] %} {% endfor %} {% endfor %}
{{ col }}
{{ value }}

El año presentado al final del nombre de algunos columnas puede variar y puede estar incluido o no.
Sin embargo,en caso de que no esté incluido, debera eliminarse el guion bajo que precede a dicho año.



Formato dataset de reentreno:


Eldataset de reentreno sigue le mismo formato que el dataset de predicción por csv.
La única variación es que debe contener una columna adicional que será el target.
Esta columna debe nombrarse como "Alquiler_mes_vu_m2" ó "Alquiler_mes_vu_m2_2020",
pudiendo sustituirse el 2020 por cualquier otro año.
